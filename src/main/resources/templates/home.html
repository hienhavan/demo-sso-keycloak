About This Demo

This application demonstrates the implementation of Single Sign-On (SSO) using Keycloak with Spring Boot. It follows Domain-Driven Design principles and showcases how to integrate authentication and authorization in a modern web application.

Key features demonstrated:

  • OAuth2/OpenID Connect integration with Spring Security
  • JWT token validation and processing
  • Role-based access control
  • User synchronization between Keycloak and application database
  • Secure logout handling
